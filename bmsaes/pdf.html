PDF Files

PDF (Portable Document Format) is a file format which should make documents in that format readable under any operating system which has software to read them. This software is usually the free Acrobat Reader, available from Adobe Systems.

Use of links and bookmarks in BMSAES

PDF files in BMSAES can take advantage of the linking and cross-referencing facilities available in the PDF format:

  • Clicking on a figure reference will jump to that figure
  • Clicking on an underlined web link will take you to that web page. If this does not work, it probably means your web browser is not properly configured
  • a bookmark structure for navigating around the document is in each file, but by default is not visible. Use Show Bookmarks on the Window menu to see them
Problems reading files

BMSAES PDF files require at least Acrobat Reader version 4 or above. If you receive error messages when opening a file, first please ascertain whether you are using version 3; if so please upgrade.

When opening a link to a PDF file in a web browser, the browser may be configured to open PDF in one of two ways. One is to use a browser plug-in for Acrobat, and the other is to open Acrobat Reader as 'helper application' and view the file in that way. We have noted in the BM that the plug-in is not always capable of reading the PDF files we and others generate, especially those with complex bookmarks in them. If you receive an error when trying to open a file, we suggest that you either use an option to save the file and then open it separately with Acrobat Reader, or reconfigure your browser to open PDF with Reader rather than the plug-in.

Some of the graphics in the screen-resolution versions of the files on this site may be less than clear when viewed at a large magnification. In this case we suggest you view the printer-resolution version. Compromises with graphics resolution have to be made to obtain the relatively small file sizes for the screen-resolution files.
